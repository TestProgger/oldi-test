# oldy-test

## Instructions for launching the project

## Frontend

### For *nix
```bash

    >>> cd frontend/
    >>> npm install && npm run start

```

### For Windows
```bash

    >>> cd frontend/
    >>> npm install ; npm run start

```

## Backend

### For *nix
```bash

    >>> cd backend/
    >>> npm install && npm run start

```

### For Windows
```bash

    >>> cd backend/
    >>> npm install ; npm run start

```
